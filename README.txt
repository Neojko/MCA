Make a .jar and then enter this :

java -jar mca.jar where_to_read where_to_write what_to_do time_limit_seconds

where : 
- where_to_read is the .graph file in the data.
- where_to_write is where you gonna write your results.
- what_to_do = c or in2 according to the algorithm you want to execute.
- time_limit_seconds is .. the timeout for the algorithm in seconds.
